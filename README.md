# 📊 Road Accident Data Analysis (Excel Dashboard)

## 1. Project Title / Headline

🚦 **Road Accident Data Analysis Dashboard – Excel Analytics Project**

An interactive **Microsoft Excel dashboard** designed to analyze road accident patterns, casualty distribution, and accident conditions using pivot tables, charts, and dynamic filters. The dashboard provides insights into accident trends across **road types, road conditions, and time-based patterns** to support road safety analysis.

---

# 2. Short Description / Purpose

The **Road Accident Excel Dashboard** is a data analysis project built using Microsoft Excel to explore accident data and identify key patterns affecting road safety.

The dashboard allows users to analyze accident data through **interactive charts, pivot tables, and trend analysis**, helping to uncover insights such as accident frequency, casualty distribution, and the impact of road conditions.

This project demonstrates how **Excel can be used as a powerful data analysis tool** for creating meaningful insights from raw datasets.

The dashboard can be useful for **traffic authorities, safety analysts, researchers, and students studying accident trends.**

---

# 3. Tech Stack

The project was built using the following tools and techniques:

* **📊 Microsoft Excel** – Main platform used for data analysis and dashboard creation
* **📈 Pivot Tables** – Used to summarize accident and casualty data
* **📊 Pivot Charts** – Used to visualize accident patterns and comparisons
* **📉 Data Visualization** – Bar charts, line charts, and trend charts for insights
* **🧮 Excel Formulas** – Used for calculations and derived metrics
* **🎛️ Interactive Dashboard Design** – Structured layout for clear analysis

---

# 4. Data Source

The dataset used in this project contains **road accident data including casualty counts and accident conditions**.

The dataset includes variables such as:

* Number of casualties
* Road type
* Road condition
* Accident trends by time
* Monthly accident patterns
* Traffic environment information

The dataset was analyzed using **Excel pivot tables and calculations** to generate meaningful insights and visualizations.

---

# 5. Features / Highlights

### Business Problem

Road accidents are a major global issue leading to injuries, fatalities, and economic losses. While large volumes of accident data are collected, it can be difficult to quickly identify patterns and trends without proper visualization.

Key questions include:

* Which **road types** experience the most accidents?
* How do **road conditions** impact accident severity?
* Are there **monthly trends** in accident occurrences?
* What factors contribute most to **casualties in road accidents**?

---

# Goal of the Dashboard

The main objective of this dashboard is to:

* Transform raw accident data into **clear and meaningful insights**
* Identify **high-risk road types and conditions**
* Analyze **accident trends over time**
* Help users understand patterns that can improve **road safety awareness and planning**

---

# Walkthrough of Key Visuals

### Key Metrics & Summary

The dashboard presents summarized accident data including:

* Total casualties recorded
* Accident distribution across different road types
* Overall accident statistics for analysis

---

### Accident Analysis by Road Type

A pivot chart that compares accident casualties across different road types such as:

* Single carriageway
* Dual carriageway
* One-way street
* Slip roads

This visualization helps identify which road environments experience **higher accident risk**.

---

### Road Condition Analysis

This chart shows how different **road conditions** affect accident occurrences, including:

* Dry roads
* Wet or damp roads
* Snow or icy roads

It helps determine how environmental conditions influence accident severity.

---

### Monthly Accident Trend

A **trend chart** showing accident or casualty patterns across different months.

This helps identify:

* Seasonal accident trends
* High-risk periods during the year
* Fluctuations in accident frequency.

---

### Pivot-Based Data Analysis

The project includes a dedicated **data analysis sheet** using pivot tables to:

* Aggregate accident data
* Calculate total casualties
* Compare accident factors across categories
